Bonjour, Nom d'utilisateur!

Comment proposer des règles :

  • Pour proposer des règles, tapez dans le chat : /proposeRules survive [nombre de voisins pour survivre] birth [nombre de voisins pour naître]
  • Exemple : /proposeRules survive 2 3 birth 3
  • Pour approuver les règles proposées, tapez : /approveRules
